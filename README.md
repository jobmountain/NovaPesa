## What is NovaPesa?

NovaPesa is an AI-powered personal finance platform that gives everyday people access to the kind of financial guidance previously reserved for the wealthy.

In under 2 minutes, any user gets:

- A **NovaScore** — a personal financial health score from 0–100
- A **personalized 90-Day Money Plan** generated by AI
- A **24/7 AI Financial Advisor** that knows their full financial profile
- **Courses** to build financial knowledge and boost their score
- **Net Worth Tracker** to monitor their wealth growth
- **Spending Coach** with AI-powered tips to cut expenses
- **Referral System** to grow their network and earn points

## Tech Stack

**Frontend**
- HTML5, CSS3, Vanilla JavaScript
- Hosted on GitHub Pages + Netlify

**Backend**
- Node.js + Express
- OpenAI GPT-4o-mini API
- Deployed on Render
- Rate limiting + CORS security

**Architecture**
```
User (GitHub Pages / Netlify)
            ↓
        Frontend JS
            ↓
  Render Backend (Node.js)
            ↓
        OpenAI API
            ↓
    Real AI Response 🤖
```
